# spotify-yt-mp3
Downloads MP3 version of the songs from YouTube from the liked songs playlist of your spotify account

### Note:
1. First create a new app in developers.spotify.com/dashboard.
2. Use your client id and client secret in app.py file 
4. Apart from the packages from setup.py you will need to install ffmpef and ffprobe to download songs continuously
5. Avoid using pip to install these and download from https://www.gyan.dev/ffmpeg/builds/ ffmpeg-git-full.7z and extract only ffmpeg.exe and ffprobe.exe into the scripts folder of your virtual environment.
